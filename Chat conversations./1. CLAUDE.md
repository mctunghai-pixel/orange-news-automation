# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Project Overview

**Orange News** is a Mongolian social media news page (Facebook/Instagram). The automation pipeline fetches English-language tech/finance/crypto news, scores it for relevance, then translates and reformats it into Mongolian-language Facebook posts using the Claude API.

## Pipeline Architecture

Two-phase Python pipeline in `Automation/`:

```
Phase 1: orange_rss_collector.py
  → Fetches 10 RSS feeds (tech, finance, crypto, AI)
  → Scores each article (keyword boost/penalty + recency + source weight)
  → Outputs top_news.json (top 10 articles)

Phase 2: orange_translator.py
  → Reads top_news.json
  → Calls Claude API (claude-opus-4-5) for each article
  → Translates English → Mongolian in Orange News tone
  → Outputs translated_posts.json (ready for thumbnail/publishing)
```

## Running the Pipeline

Install dependencies first:
```bash
pip install feedparser httpx python-dateutil anthropic
```

Run Phase 1 (RSS collection):
```bash
cd Automation
python orange_rss_collector.py
# Outputs: top_news.json
```

Run Phase 2 (Claude translation):
```bash
cd Automation
python orange_translator.py
# Reads: top_news.json → Outputs: translated_posts.json
```

Requires `ANTHROPIC_API_KEY` environment variable set for Phase 2.

## Output Format

`translated_posts.json` — array of post objects:
```json
{
  "headline": "...",         // Mongolian headline
  "post_text": "...",        // 150–250 word Mongolian post body
  "hashtags": ["#OrangeNews", "..."],
  "category": "tech|finance|crypto|AI",
  "key_numbers": ["$87,000", "+2.5%"],
  "original_url": "...",
  "original_title": "...",
  "score": 8.5
}
```

## Content Style (Orange News Tone)

Posts must follow the Orange News editorial style defined in `ORANGE_NEWS_SYSTEM_PROMPT` in `orange_translator.py`:
- Positive, accessible tone — no sensationalism
- Inverted pyramid structure: headline → one-liner → 3–4 key facts → relevance for Mongolian readers → source
- 150–250 words, Cyrillic Mongolian script
- Preserve numbers/percentages/brand names exactly ($87,000, NVIDIA, Bitcoin)
- End with hashtags: `#OrangeNews #[CategoryHashtag]`
