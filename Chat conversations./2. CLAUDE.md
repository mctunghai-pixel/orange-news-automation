## GitHub Actions workflow (бэлэн болгох)
```yaml
# .github/workflows/orange_news.yml
name: Orange News Daily Pipeline
on:
  schedule:
    - cron: '0 0 * * *'  # UTC 00:00 = Монгол 08:00
  workflow_dispatch:

jobs:
  run-pipeline:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - uses: actions/setup-python@v5
        with:
          python-version: '3.11'
      - name: Install dependencies
        run: pip install anthropic feedparser httpx certifi python-dateutil yfinance Pillow requests
      - name: Run pipeline
        env:
          ANTHROPIC_API_KEY: ${{ secrets.ANTHROPIC_API_KEY }}
          FB_PAGE_ID: ${{ secrets.FB_PAGE_ID }}
          FB_ACCESS_TOKEN: ${{ secrets.FB_ACCESS_TOKEN }}
          PYTHONIOENCODING: utf-8
        run: python3 orange_translator.py && python3 image_generator.py
```
